# BUAS CMGT PR Intake
This is the project that I will assign for my intake procedure at BUAS.

The project is a simple `Asteroids` game, where you aim to survive as long as possible.

## Controls
* W, S, Up Arrow, Down Arrow - Thrust your spaceship up and down
* A,D, Right Arrow, Left Arrow - Rotate your spaceship left and right
* Spacebar - Fire a projectile
